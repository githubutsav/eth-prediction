# Crypto Price Prediction

Author: Utsav Singh
Description:
This project uses machine learning and time series analysis to predict cryptocurrency prices. We have implemented a model that uses LSTM (Long Short-Term Memory) neural networks to predict the future prices of a cryptocurrency based on historical price data.
